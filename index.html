    Introduction

    Firmware
    STM32

    M0開發板提供豐富的硬體接口和處理能力，控制與運用各類感測器收集環境數據。

    Sensor

    透過各種感測器收集環境數據，如溫度、濕度、距離等等。並與STM32連接

    Raspberry Pi

    使用Mjpg-streamer與Django分別提供網路Camera畫面與網路接收指令的伺服器端

    APP
    Android Studio

    主要以Kotlin進行開發，介面設計以直覺性和美觀性為主，提升使用者體驗。

    Code

    透過使用Fragment增加APP效能，也便於維護和擴展。

    Firebase

    擁有使用者登入、註冊、個人資料顯示與修改介面，並透過Firebase管理使用者資料。
